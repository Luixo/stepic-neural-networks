Задачи курса "Нейронные сети" на stepic.org (neural networks course excercises)
====================

## Структура
Файлы отсортированы по папкам в зависимости от модуля и названы в соответствии с номером урока и номером степа в уроке (например, 1.6.5), а также имеют краткое название.

## Использование
Для запуска кода используется команда
```
python filename.py < filename.txt
```
В случае возникновения ошибки No module named 'numpy' установите корректные переменные окружения, включающие в себя эту библиотеку (или скачайте WinPython и запускайте питон с его помощью).

## Проверка задач
Код может разделяться комментариями на debug и production версию, первая предназначена для подготовки ко второй. Если код на разделен - то он весь является production версией.
